# Instruções do projeto

``` yaml
Considere o array [3, 7, 9, 1, 0].

O primeiro elemento inserido foi o 3, o segundo foi o 7 e assim por diante, seguindo a ordem.

Levando em conta as três estruturas de dados (fila, lista e pilha) e as suas regras de operações, qual seria a sequência dos números removidos em cada uma delas para esvaziá-las completamente utilizando o array mencionado?
```

- ## Resposta:


Sequência dos valores a serem removidos em cada uma das estruturas de dados comentadas respectivamente.
  - **Fila:** 3, 7, 9, 1, 0 `(método: shift() - Desenfileiramento)`
  - **Lista:** Indefinido. Não existe uma regra que especifique uma ordem de remoção de elementos neste tipo de estrutura de dados. `(método: splice() - Remoção)`
  - **Pilha:** 0, 1, 9, 7, 3 `(método: pop() - Desempilhamento)`
